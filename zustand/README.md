## Running project

To the step below work is necessary that all developer environment be correctly configured.

I'm using **npm** to install the dependencies of the project in powershell terminal.

Step 1:

```cl
npm i
```

Step 2:

```cl
npm run dev
```

Step 3:

```cl
npm run server
```

Testing:

```cl
npm run test
```
